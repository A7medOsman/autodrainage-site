How it works

  1. Set design criteria — choose Storm or Foul, set ks & viscosity (UK-ready defaults).
  2. Build network — add pipes, inflows, and downstream references (non-sequential allowed).
  3. Run & review — check capacity and flags against DCG/CEWIS thresholds.
  4. Export — download tidy PDFs/CSVs for submissions and reviews.
